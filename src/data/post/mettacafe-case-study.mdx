## Overview

Metta Cafe is a digital transformation project designed to bring a unique Munich venue's three-fold offering - caf&eacute;, bakery, and yoga studio - into the digital space. The project delivers both a website and mobile app that helps health-conscious customers discover products, book classes, and engage with the venue's community.

| **Role**       | **Details**                                                                    |
| -------------- | ------------------------------------------------------------------------------ |
| **Team**       | Lukas Lehmann & Beyza &Ouml;zsel as UX Designers                               |
| **Users**      | Health-conscious individuals, yoga enthusiasts, dietary-restricted customers   |
| **Peers**      | WBS Coding School mentors and fellow students who provided feedback            |
| **Timeframe**  | Three sprints: Aug 5-9, Sept 9-13, Sept 30-Oct 4, 2024                         |
| **Tools Used** | Figma, Google Forms (for surveys), Slack (for interviews and usability testing |
