## Overview

FocusBuddy for macOS is designed to help individuals, especially those with ADHD or productivity challenges, stay on task and accountable during work sessions. With a clean, intuitive interface, FocusBuddy provides a supportive space where users can find accountability partners or groups and use tools like the Pomodoro timer and focus-enhancing music to boost productivity.

| **Aspect**            | **Details**                                                                                                                                                             |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Role**              | Led end-to-end product design, including UX/UI design, user research, and testing.                                                                                      |
| **Team**              | Individual project with collaboration from the ADHD community and remote workers for feedback.                                                                          |
| **Timeframe**         | October 21st - November 8th, 2024.                                                                                                                                      |
| **Inciting Incident** | Many individuals struggle with focus, time management, and feelings of isolation while working alone. This inspired the creation of a tool to address these challenges. |

## Challenges Identified

- Difficulty maintaining focus on tasks.
- Struggles with time management.
- Need for structured routines and accountability.
- Feelings of loneliness and isolation.

## Design Process

1. **Research**: Conducted surveys and interviews with individuals with ADHD and productivity challenges.
   {/* ![Survey Results](../assets/images/SurveyResults.png "Survey results highlighting user pain points") */}
2. **Ideation**: Brainstormed features like accountability partners, timers, and focus-enhancing music.
   {/* ![Brainstorming Sketches](../assets/images/BrainstormingSketches.png "Early brainstorming sketches of features") */}
3. **Prototyping & Testing**: Created low-, mid-, and high-fidelity wireframes; tested designs with users to refine the interface.
   {/* ![Wireframes](../assets/images/Wireframes.png "Low-fidelity wireframes created during prototyping phase") */}

---

{/* Climax */}

## Pivotal Moments

- During user testing, feedback from neurodivergent users emphasized the importance of customization and creating a safe environment. This insight led to prioritizing features like dark/light mode and flexible timer settings.
- A significant decision was made to integrate focus-enhancing music directly into the app after discovering its positive impact on productivity.

{/* ![High-Fidelity Prototype](../assets/images/HighFidelityPrototype.png "High-fidelity prototype showcasing the Pomodoro timer feature") */}

---

{/* Falling Action */}

## Final Iterations

- Refined the interface for macOS-specific design elements.
- Implemented key features such as:
  - Pomodoro timer for structured work sessions.
  - Shared agendas for collaboration.
  - Chat features for real-time communication with accountability partners.
  - Focus sounds/music integration to enhance concentration.

{/* ![Final Design Screenshots](../assets/images/FinalDesign.png "Final design showcasing key features like the timer and shared agenda") */}

User testing confirmed that the app provided a supportive space for focus and accountability.

---

{/* Resolution */}

## Outcomes

- Created a user-friendly app that offers a safe space for focus and accountability.
- Learned macOS-specific design principles and discovered new Figma plugins for building a design system.
- Identified opportunities to enhance customization for diverse user needs.

## Future Improvements

- Expand customization options for timers and focus sessions.
- Broaden the library of focus sounds/music.
- Improve mobile optimization for cross-device use.
- Integrate task management tools like Apple Reminders, Trello, or Asana.

{/* ![Outcome Metrics](../assets/images/Outcomes.png "User feedback highlights showing success metrics") */}

**This project taught me the importance of designing inclusive solutions that cater to neurodiverse users. It also reinforced my commitment to creating safe digital spaces that foster productivity and collaboration.**
