Color Contrast Checker

This tool helps you verify which colors in your theme's palette meet the WCAG contrast requirements for accessibility in both light and dark modes.

Understanding Contrast Ratings

  • 7.0+ AAAExcellent contrast, exceeds WCAG AAA (7:1)
  • 4.5+ AAGood contrast, meets WCAG AA for normal text (4.5:1)
  • 3.0+ AA LargeFair contrast, meets WCAG AA for large text only (3:1)
  • FailPoor contrast, doesn't meet minimum WCAG requirements
